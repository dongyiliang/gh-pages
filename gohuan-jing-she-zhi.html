go环境设置

yiliang.dong 五月 11, 2022 环境配置

下载地址

https://golang.google.cn/dl/

下载版本

go1.18.2.windows-amd64.zip

解压路径

D:\apps\go\go1.18.2

设置GOROOT变量

GOROOT=D:\apps\go\go1.18.2 在PATH变量中追加GOROOT %GOROOT%\bin

新建gospace目录，

存放gopath工作目录、goenv虚拟环境、gocache缓存

GOPATH=D:\gospace\gopath
GOCACHE=D:\gospace\gocache
GOENV=D:\gospace\goenv·

GOPATH是后面go语言的工作目录，在里面新建src目录，用于存放代码
src // 存储go的源代码（需要我们自己手动创建）
pkg // 存储编译后生成的包文件 （自动生成）
bin // 存储生成的可执行文件（自动生成）

国内提速

GOPROXY=https://goproxy.cn,direct

新建文件gopath/src/test.go

   fmt.Println("Hello, World!")
}

执行测试

D:\gospace\gopath\src>go run test.go
